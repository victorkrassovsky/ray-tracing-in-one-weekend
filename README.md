# ray-tracing-in-one-weekend
A ray tracing graphics engine. 
Going through this [book](https://raytracing.github.io/books/RayTracingInOneWeekend.html)

## Features:
- Geometric objects:
    1. Spheres
    2. Triangles (pretty much all you need)
- Materials:
  1. Lambertian
  2. Metal
  3. Dialectric (refracts)
- Diffusion
- Antialiasing
- Camera mobility
